# sns_ai_automation_agency_web

sns_ai_automation_agency の WebUI を構築

## 手順

### 1.環境構築

node.js をインストール

- https://nodejs.org/ja/download

### 2.プロジェクト作成

```shell
npx create-next-app@latest scene_image_picker --ts --tailwind
```

### 3.ローカル起動

```shell
cd scene_image_picker && npm run dev
```

### 4.Vercel デプロイ

Git と連携しているため、マージするとデプロイされる

[Vercel](https://vercel.com/sns-ai-automation-agency-pocs-projects)
