# 前提スキル (Base Skills)

このスキルを実行する際は、**必ず `python-pro` スキルを同時に参照し**、そこで定義されたコーディング規約（型ヒント、Docstring、pathlib使用など）を遵守してください。

# デザイン指針

1. **設計書の受け取り:** ユーザーから設計書が与えられます。内容を理解してください。設計書が見当たらない場合は、設計書を用意してほしい旨をユーザーに依頼します。
2. **内容理解:** 設計書の内容を理解します。不明点があればユーザーに質問してください。
3. **方針承認:** 実装方針をユーザーに提示します。提示内容は簡潔にまとめてください。方針が複雑な場合はタスクに分解してください。
4. **承認と実装:** ユーザーから「OK」と承認が得られたら、実装を始めてください。必ず承認されてから実装をしてください。

# 実装方針の形式

- 概要: 簡潔な要約
- 成果物: 編集・作成されるとディレクトリ構造
- クラスやメソッドの単位の簡単な説明
