# Phase 1: 地形の基本 ― 形を作る力

**目標:** リファレンス画像（亜熱帯の島・海岸線）の地形を作れる。Sculpt Modeで意図した形に調整できる。
**完了条件:** A.N.T. → Sculpt → Ocean Modifierの一連のワークフローで、リファレンスに近い地形シルエットを作れる

**リファレンス画像の地形要素:**
- 右側: 緑に覆われた丘陵（なだらかな山）
- 中央: 白い砂浜の海岸線（緩やかなカーブ）
- 手前: 岩場が点在する浅瀬
- 左側: 深い青の海
- 遠景: 低い島影

---

## 1-1. A.N.T. Landscape でベース地形

### 有効化手順
1. 編集 → プリファレンス → アドオン
2. 「A.N.T. Landscape」を検索して有効化
3. `Shift-A` → メッシュ → Landscape

### 主要パラメータ
- **Noise Type**: 地形パターン（Perlin, Voronoi等）
- **Noise Size**: 地形の大きさ（小さい値 = 細かい起伏）
- **Height**: 高さのスケール
- **Subdivisions**: メッシュの解像度（高いほど細かい）
- **Random Seed**: 乱数シード（値を変えるとパターンが変わる）

### やること
1. デフォルトCubeを削除
2. `Shift-A` → メッシュ → Landscape
3. 左下のパネルでパラメータを調整
4. seed値を変えて複数パターン試す
5. リファレンスの丘陵に近いシルエットを探す

---

## 1-2. Sculpt Mode で形状調整

A.N.T.で「だいたい良い形」を作ったら、Sculpt Modeで意図通りに調整する。**このステップが不可欠。**

### Sculpt Modeに入る前に
- `Ctrl-A` → 全トランスフォーム を適用（スケールを1にする）
- Multiresolutionモディファイアを追加してSubdivide（解像度を上げる）

### 必須ブラシ5つ

| ブラシ | 用途 | コツ |
|--------|------|------|
| **Draw** | 盛り上げ・掘り下げ | 基本。`Ctrl`押しで逆方向 |
| **Grab** | 掴んで大きく移動 | シルエットの調整に最適 |
| **Smooth** | 滑らかにする | **`Shift`押しで常時発動**（最重要） |
| **Flatten（Plane）** | 平坦化 | 台地・平らな面を作る |
| **Clay Strips** | 粘土を盛る | 体積を積極的に構築。ラフパス向き |

### ワークフロー
1. **Grab**で大きなシルエットを調整（山の位置、海岸線のカーブ）
2. **Clay Strips / Draw**で体積を盛る・削る
3. **Smooth**（`Shift`押し）で滑らかに
4. **Flatten**で必要な平面を作る

### やること（リファレンス画像に合わせて）
- 右側を高くして丘陵に
- 手前〜中央を低くして海岸線に
- 海岸線は緩やかなカーブを描くように

---

## 1-3. Displace Modifier で地形ディテール

Sculptで大まかな形を作った後、テクスチャベースで細かい凹凸を追加。

### 手順
1. サブディビジョンサーフェス（Simple）を追加 → レベル2〜3で解像度を上げる
2. ディスプレイスモディファイアを追加
3. 「新規」でテクスチャを作成 → テクスチャプロパティで「クラウド」等を設定
4. **Strength**: 小さい値（0.1〜0.3）で微細な凹凸

### 主要パラメータ
- **テクスチャ**: 変位の元（クラウド、ノイズ等のプロシージャルテクスチャ）
- **方向**: Normal（法線方向）が自然
- **強度（Strength）**: 変位量。マイナスで反転
- **中間レベル（Midlevel）**: 0.5がデフォルト。この値が「変位ゼロ」

---

## 1-4. Ocean Modifier で海面

地形メッシュの下にPlane（Z=0付近）+ Ocean Modifierで海を作る。

### 手順
1. `Shift-A` → メッシュ → 平面
2. スケールを大きくして海面サイズに
3. モディファイアー → 物理演算 → Ocean を追加

### 主要パラメータ

| パラメータ | 説明 | 推奨値 |
|-----------|------|--------|
| **ジオメトリ** | Generate（グリッド生成）/ Displace（既存メッシュ変形） | Generate |
| **解像度** | 内部グリッド（高いほど細かい波） | Viewport: 12, Render: 16 |
| **Time** | アニメーション用。キーフレームで動く | 静止画なら任意の値 |
| **Depth** | 海底の深さ。浅いと細かい波 | 200 |
| **Scale** | 波の振幅 | 1.0前後 |
| **Choppiness** | 波頭の鋭さ | 1.0前後 |
| **Wind Velocity** | 風速。低い=穏やかな海 | 3〜5（穏やかな海） |

### Blender 5.0 API注意 `[実験済]`
- `wind_velocity`（旧`wind_speed`）
- `aerosol_density`（旧`dust_density`）

## 1-5. Geometry Nodes 入門（概念のみ）

Phase 1では深入りしない。以下の概念だけ理解する:
- Set Position + Noise Textureで頂点の高さを制御できる
- ノードベースのプロシージャル地形生成が可能
- Phase 3の散布（Instance on Points）の基礎になる

---

## Phase 1 作業順序

```
Step 1: A.N.T. Landscapeでベース地形を生成
Step 2: Sculpt Modeでリファレンスに合わせてシルエット調整
Step 3: Displace Modifierで微細な凹凸追加
Step 4: 別のPlane + Ocean Modifierで海面配置
Step 5: ビューポートで全体確認 → 保存
```
