# Phase 3: 自然要素 ― 世界を埋める力

**目標:** 地形に植生・岩・水の要素を追加して「世界」にできる。

---

## 3-1. Instance on Points（散布の基本） `[実験済]`

### 概念

地形の表面にオブジェクト（木、草、岩など）をランダムに配置する技術。

### Geometry Nodesの構成

```
グループ入力 [Geometry] ─┬→ 面にポイント配置 [Mesh]
                         │
                         └→ ジオメトリ統合 [Geometry]（地形の表示用）

面にポイント配置 [Points] → ポイントにインスタンス作成 [Points]
オブジェクト情報 [Geometry] → ポイントにインスタンス作成 [Instance]

ポイントにインスタンス作成 [Instances] → ジオメトリ統合 [Geometry]
ジオメトリ統合 [Geometry] → グループ出力 [Geometry]
```

### 重要なポイント

1. **ジオメトリ統合が必要**: 地形メッシュとインスタンスを両方出力するために、Join Geometryノードで統合する。なければ地形が消える
2. **グループ入力から2本の線**: 面にポイント配置（散布対象）とジオメトリ統合（表示用）の両方に接続
3. **オブジェクト情報でインスタンス元を指定**: 散布するオブジェクトを選択する

### 主要パラメータ

| パラメータ | ノード | 効果 |
|-----------|--------|------|
| 密度 | 面にポイント配置 | 散布密度（高い=多い、低い=少ない） |
| シード | 面にポイント配置 | ランダムパターンを変える |
| スケール | ポイントにインスタンス作成 | インスタンスのサイズ |
| 回転 | ポイントにインスタンス作成 | インスタンスの向き |

### ハマりポイント `[実験済]`

- **インスタンス元オブジェクトの原点が重要**: オブジェクトの原点がずれていると、インスタンスが意図しない位置に表示される（今回ICO球のZ座標が206にあったため、インスタンスが空中に配置された）
- **オブジェクトのスケールと地形のスケールの比率**: 地形が大きい場合、インスタンス元オブジェクトも相応のサイズが必要
- **密度の調整**: 地形の面積に依存するので、まず低い値（0.01）から始める
