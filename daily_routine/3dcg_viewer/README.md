# 3DCG Viewer

Webベースの3Dビューワー納品による建築パース市場への参入。

## コンセプト

ココナラ等の建築パース市場では、3DCGで作成したモデルを静止画像で納品するのが主流。
これをインタラクティブな3Dビューワーとして納品し、顧客が任意のアングルから空間を閲覧できるようにする。

## 差別化要因

- 3DCG（Blender）× エンジニアリング（Three.js/glTF）の掛け算
- 競合は建築士・デザイナーでありエンジニアではない。Web技術でのデプロイは彼らの守備範囲外

## 技術構成

```
間取り図/図面 → Blender（3Dシーン構築） → glTF/GLB出力 → Webビューワー
```
