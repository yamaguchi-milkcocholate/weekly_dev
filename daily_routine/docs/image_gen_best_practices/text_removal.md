# テキスト/オーバーレイ除去 ベストプラクティス

## 概要

seed 画像（vlog キャプチャ）上の時刻表示・テロップ等のテキストオーバーレイを自然に除去する。

## 推奨手法

### API・方式

- **Gemini 3 Pro**（`gemini-3-pro-image-preview`）
- **Python SDK**（`google-genai`）
- **I2I 編集**（seed 画像 1 枚 + テキスト指示）
- 参照画像は不要

### 推奨プロンプト

```
Remove all text overlays from this image.
```

これだけで十分。追加の保持指示・コンテキスト説明は不要。

## 注意点・アンチパターン

### やってはいけないこと

| アンチパターン                                               | 理由                                                     |
| ------------------------------------------------------------ | -------------------------------------------------------- |
| 除去対象のテキスト内容を具体的に記述する                     | セーフティフィルタに引っかかる（特に「Japanese text」）  |
| ALL CAPS 保持指示（MUST/MUST NOT）を追加する                 | 成功率が下がる。テキスト除去では不要                     |
| 「Reconstruct the underlying content」等の復元指示を追加する | フィルタトリガーのリスク増。モデルは指示なしでも復元する |

### 知っておくべきこと

- **顔匿名化マスク（ハート・横棒）は除去されない**: Gemini はこれらを「テキスト」ではなく画像の一部として扱う。前処理として安全
- **背景復元品質は高い**: テキストが人物や複雑な背景に重なっていても自然に復元される
- **コスト**: $0.04/枚
