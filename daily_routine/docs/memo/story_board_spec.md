# 絵コンテ記述要件

絵コンテに記載すべき情報の要件を定義する。

---

## 1. 全体情報

| 項目        | 必須 | 説明                              | 例                                                                                                             |
| ----------- | ---- | --------------------------------- | -------------------------------------------------------------------------------------------------------------- |
| タイトル    | ○    | 動画タイトル（日本語）            | AIインフルエンサー案件 - コーヒーCM                                                                            |
| 総尺        | ○    | 動画全体の秒数                    | 15                                                                                                             |
| BGMの方向性 | ○    | BGM調達の指示（日本語、自然言語） | おしゃれなカフェBGM、BPM100前後、Lo-fi Hip Hop系。CUT03でビートドロップ、CUT06以降は余韻を残してフェードアウト |

---

## 2. キャラクター定義

動画に登場する全キャラクターを定義する。キャラクターごとに以下を記載。

| 項目       | 必須 | 説明                                           | 記述言語 |
| ---------- | ---- | ---------------------------------------------- | -------- |
| 名前       | ○    | キャラクターの識別名                           | 日本語   |
| 外見       | ○    | 年齢・性別・髪型・髪色・肌色・体型・顔の特徴等 | **英語** |
| 服装       | ○    | 衣装の詳細（色・素材・アイテム等）             | **英語** |
| 登場カット | ○    | どのカットに登場するか                         | -        |

### 記述例

```
名前: みく
外見: Japanese woman in her early 20s, long straight black hair with side-swept bangs,
      slim build, soft round face, large expressive eyes
服装: white oversized knit sweater, light blue denim jeans, white sneakers
登場カット: CUT 02, CUT 06, CUT 07
```

### 注意事項

- 外見・服装は**英語**で記述する（画像生成プロンプトに直接使用されるため）
- キャラクターの一貫性はこの定義に依存する。曖昧だと各カットで別人になる
- カットごとに衣装が変わる場合はカット情報側で上書きを指定する

---

## 3. カット情報

各カットについて以下を記載する。

### 3.1 基本情報

| 項目           | 必須 | 説明                 | 制約             |
| -------------- | ---- | -------------------- | ---------------- |
| カット番号     | ○    | 連番                 | 1始まり          |
| 尺（秒）       | ○    | カットの長さ         | **2〜5秒の整数** |
| キャラ登場     | ○    | キャラクターが映るか | true / false     |
| トランジション | ○    | 次カットへの繋ぎ方   | 下記参照         |

#### トランジションのルール

| 値           | 用途                         |
| ------------ | ---------------------------- |
| `fade_in`    | 動画冒頭（最初のカットのみ） |
| `cut`        | 同一シーン内のカット切り替え |
| `cross_fade` | シーンが変わる最初のカット   |
| `fade_out`   | 動画末尾（最後のカットのみ） |

### 3.2 映像の描写

| 項目         | 必須 | 説明                                     | 記述言語 |
| ------------ | ---- | ---------------------------------------- | -------- |
| 状況説明     | ○    | そのカットで何が起きているかの概要       | 日本語   |
| 背景描写     | ○    | **キャラクターを除いた**背景・環境の描写 | **英語** |
| 被写体の動作 | ○    | そのカットで行われる**1つの**アクション  | 日本語   |
| ポーズ指示   | △    | キャラの姿勢・ポーズの具体的指示         | 自由     |

#### 背景描写の注意事項

- Asset Generator が**人物なしの背景画像**を生成するために使用する
- 「カフェの窓際」ではなく、具体的に描写する：

```
# NG
カフェの窓際

# OK
Cozy cafe interior, wooden table by a large window, warm afternoon sunlight
streaming through glass, potted plant on windowsill, blurred city street
visible outside, soft warm color palette
```

#### 1カット1アクションの原則

I2Vモデル（Runway Gen-4 Turbo）は**1カットにつき1アクション**が最適。複数の動作がある場合はカットを分割する。

```
# NG（1カットに複数アクション）
ライトの位置調整 → コーヒーカップの配置 → カメラ構図決め

# OK（3カットに分割）
CUT 02a: ライトの位置を調整する
CUT 02b: コーヒーカップをテーブルに置く
CUT 02c: カメラのファインダーを覗き込む
```

### 3.3 カメラワーク

| 項目       | 必須 | 説明                             | 記述言語 |
| ---------- | ---- | -------------------------------- | -------- |
| 種別       | ○    | カメラワークの分類               | 英語     |
| 動きの詳細 | ○    | 具体的なカメラの動き             | **英語** |
| 動きの強度 | ○    | 被写体とカメラを含む全体の動き量 | 下記参照 |

#### カメラワーク種別

| 値                  | 説明                     |
| ------------------- | ------------------------ |
| `wide`              | 全景・引きの画           |
| `medium`            | 腰〜上半身程度           |
| `close-up`          | 顔やオブジェクトのアップ |
| `extreme-close-up`  | 手元・目元等の超接写     |
| `POV`               | 一人称視点               |
| `over-the-shoulder` | 肩越しショット           |
| `follow`            | 被写体を追うカメラ       |
| `overhead`          | 俯瞰・真上からの視点     |

#### 動きの詳細の記述例

```
slow zoom-in from medium to close-up
static tripod shot, eye level
slow pan left to right revealing the table
camera slowly dollies forward
handheld slight movement, intimate feel
```

#### 動きの強度

| 値         | 説明                           | 例                             |
| ---------- | ------------------------------ | ------------------------------ |
| `STATIC`   | カメラワークのみ、被写体は静止 | 静物のクローズアップ           |
| `SUBTLE`   | 微細な動き                     | 髪揺れ、湯気、瞬き             |
| `MODERATE` | 中程度の動き                   | コーヒーを飲む、ページをめくる |
| `DYNAMIC`  | 大きな動き                     | 歩く、立ち上がる               |

### 3.4 テロップ

| 項目             | 必須 | 説明                                                 |
| ---------------- | ---- | ---------------------------------------------------- |
| テロップテキスト | △    | 画面に表示するテキスト（日本語）。不要なカットは空欄 |

---

## 4. 音声・SE情報

カット単位またはパート単位で記載する。

| 項目         | 必須 | 説明                        | 例                                               |
| ------------ | ---- | --------------------------- | ------------------------------------------------ |
| セリフ       | △    | 話者と台詞テキスト          | 山口さん：「このコーヒーのCM作ってくださいよー」 |
| SE（効果音） | △    | 具体的な音の名称            | コーヒーを注ぐ音、豆が弾ける音、シャッター音     |
| BGM変化      | △    | そのカットでのBGMの状態変化 | ビートドロップ、フェードアウト開始、無音         |

---

## 5. 制約まとめ

| 制約                                | 値                                      |
| ----------------------------------- | --------------------------------------- |
| カット尺                            | 2〜5秒（整数のみ）                      |
| 1カットのアクション数               | 1つまで                                 |
| 全カット数                          | 10〜40                                  |
| 全カット合計尺                      | 総尺と完全一致                          |
| 各シーンのカット合計尺              | シーン尺と完全一致                      |
| 外見・服装・背景描写・カメラ詳細    | 英語で記述 (ただし日本語訳を下に添える) |
| 状況説明・動作・テロップ・BGM方向性 | 日本語で記述                            |

---

```markdown

- 服装: （英語）

| 項目           | 内容                                  |

```
