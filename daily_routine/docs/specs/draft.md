# プロジェクト仕様書：「〇〇の一日」AI動画生成自動化パイプライン

## 1. プロジェクト目的

- **ゴール:** YouTube ショート（9:16）形式の「〇〇の一日」動画を、一貫性を保ちながら全自動生成する。
- **ターゲット:** SNSマーケティング、ブランド広告、個人VlogのAI代替。
- **提供価値:** 競合分析から素材生成、編集までを自動化し、トレンドに即した高クオリティな動画を量産する。

## 2. システムアーキテクチャ

プラットフォーム非依存、APIおよび各種AIモデルをPythonで統合。

| レイヤー                      | 役割                                                  | 採用候補技術                                |
| ----------------------------- | ----------------------------------------------------- | ------------------------------------------- |
| **1. Intelligence Engine**    | **[新規]** トレンド分析・構成抽出（競合動画の構造化） | **YouTube API** / Whisper / GPT-4o (Vision) |
| **2. Scenario Engine**        | 抽出データに基づくストーリー・プロンプト生成          | Gemini 1.5 Pro / GPT-4o                     |
| **3. Asset Collection**       | 一貫性維持のための参照用資材（画像）の生成            | Midjourney / Stable Diffusion / DALL-E 3    |
| **4. Visual Core (V-Engine)** | 資材を参照した動画素材の生成                          | **Google Veo** / Luma / Runway Gen-3        |
| **5. Audio & Voice**          | ナレーション、環境音、効果音(SE)の生成                | ElevenLabs / Google Veo (Audio)             |
| **6. Post-Production**        | 自動テロップ、SE挿入、バッチ補正、結合                | MoviePy / FFmpeg / Faster-Whisper           |

## 3. 業務フロー（パイプライン）

1. **Market Intelligence (トレンド抽出フェーズ):**

- YouTube API等を用い、特定のキーワード（例：「OLの一日」「モーニングルーティン」）でバズっているショート動画を収集。
- **自動抽出:** \* **構成:** 何秒でシーンが切り替わるか、冒頭のフック（引き）は何か。
- **トレンド:** 流行しているBGMのテンポ、テロップのフォントや配色。
- **資材抽出:** どのような小物（コーヒー、PC、バッグ等）が映っているか。

2. **Scenario Generation:** 抽出した成功パターンに基づき、具体的な1日のシナリオとプロンプトを作成。
3. **Asset Collection (資材収集フェーズ):**

- シナリオとトレンド分析で特定された「必須資材（キャラ、服、小物、背景）」を静止画として高品質生成し、ストック。

4. **Visual Generation:** 収集した資材をリファレンスとして入力し、動画を生成。プロンプトにも分析済みのカメラワーク（POV、クローズアップ等）を反映。
5. **Audio & Post-Production:**

- 分析された「トレンドのテンポ」に合わせてカット割り。
- 音声解析による動的テロップ配置、および動作に同期した効果音の自動挿入。

## 4. 重点開発項目

- **Trend Decomposition (トレンド解体):** 既存動画を「映像・音声・メタデータ」に分解し、再構成可能なデータに変換するアルゴリズム。
- **Asset Driven Consistency:** 静止画資材を「正解データ」として動画生成AIに強く反映させるリファレンス制御技術。
- **Dynamic Captions:** 視聴維持率を最大化する、YouTube ショート特有の強調アニメーションテロップの自動配置。
- **Sound-Image Sync:** 映像内の動作と効果音（SE）をミリ秒単位で同期させる音響設計。

## 5. 編集・テロップ・SE仕様

- **フォーマット:** 1080x1920 (9:16), 30/60fps.
- **テロップスタイル:** 分析されたトレンド（バズっている動画の配色等）を自動反映。
- **効果音構成:** 映像内の「物体」や「動作」をAIが認識し、最適なSE（足音、キーボード打鍵音等）を自動ライブラリから割り当て。
